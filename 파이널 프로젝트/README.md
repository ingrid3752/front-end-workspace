## Category 값들

1. T1H 기온 (섭씨)

- fcstValue : 현재 시간의 기온

2. PTY 강수 형태

- fcstValue :
- 0 : 강수 없음
- 1 : 비
- 2 : 비/눈
- 3 : 눈
- 4 : 소나기

3. POP : 강수 확률 (%)

- fcstValue : 현재 시간의 강수 확률

4. REH : 습도 (%)

- fcstValue : 현재 시간의 습도

5. SKY : 하늘 상태 (맑음, 구름, 흐림)

- fcstValue :
- 1 : 맑음
- 3 : 구름 많음
- 4 : 흐림

6. VVV : 풍속 (남북)

- fcstValue : m/s 단위로 풍속을 나타내며, 양수는 남풍, 음수는 북풍

7. UUU : 풍속 (동서)

- fcstValue : m/s 단위로 풍속을 나타내며, 양수는 동풍, 음수는 서풍

8. WSD : 풍속 (m/s)

- fcstValue : 현재 시간의 풍속

9. TMP : 1시간 간격 기온 (섭씨)

- fcstValue : 현재 시간의 기온

10. LGT : 낙뢰

- fcstValue :
- 0 : 낙뢰 없음
- 1 : 낙뢰 있음
